Basic Grids

Dynamic grid, features, expands to fit the grid row by default, if no (viewport size) is specified, the cell will default to 100% of the row `.cell` will default to 100% unless a viewport size is specified `.cell-#`

Individual Sizing

Unlike the basic grid, specifing a size on the class `cell-sm-# will only fit that portion of the row. bsaically, it sets the width of the cell, whereas not specifing, will let the grid figure it out
